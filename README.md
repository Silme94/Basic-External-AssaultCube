## HOW TO USE
1. Download [Python](https://www.python.org/downloads/).
2. Clone the project.
```
git clone
cd
```
3. Open [Assault Cube](https://assault.cubers.net/download.html) (1.3.0.2)
4. Run the .bat file in bin folder.
```
cd bin
.\run.bat
```
